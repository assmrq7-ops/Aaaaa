# מערכת ניהול הזמנות - ארבס מובחר

מערכת ניהול מתקדמת לעסק ארבס מובחר, לניהול לקוחות, הזמנות, החזרות והוצאות.

## העלאה לנטליפי (Netlify)

### אופציה 1: גרירה ישירה של תיקיית ה-ZIP
1. היכנס ל-[Netlify](https://app.netlify.com/)
2. לחץ על "Add new site" -> "Deploy manually"
3. גרור את קובץ ה-ZIP או את התיקייה שחילצת
4. המתן לסיום הבנייה - זה ייקח כ-2-3 דקות

### אופציה 2: חיבור ל-GitHub
1. העלה את הקבצים ל-GitHub repository
2. ב-Netlify לחץ על "Add new site" -> "Import from Git"
3. בחר את ה-repository שלך
4. Netlify יזהה אוטומטית את הגדרות הבנייה מקובץ netlify.toml

## הרצה מקומית

```bash
# התקנת תלויות
npm install

# הרצה במצב פיתוח
npm run dev

# בנייה לפרודקשן
npm run build

# תצוגה מקדימה של הבנייה
npm run preview
```

## תכונות
- ניהול לקוחות מלא
- יצירת הזמנות עם מחירים שונים ללקוחות קבועים ומזדמנים
- מעקב אחר החזרות
- דוחות פיננסיים
- ייצוא לאקסל
- שמירה אוטומטית ב-localStorage
- תמיכה בעברית וממשק RTL
- ממשק מותאם לנייד

## דרישות מערכת
- Node.js 18 ומעלה
- דפדפן מודרני (Chrome, Firefox, Safari, Edge)
